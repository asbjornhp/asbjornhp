## Featured Public Academic Projects and Coursework 🤓

#### [Master Thesis: "Hypothesizing Agents: Introducing Abduction in AND/OR Graph Search"]
- **Concept:** To enhance scaleability and transparency of automated planning in environments with human interference and incomplete information, for example a warehouse or hospital, an in-depth framework based on dynamic epistemic logic was concieved and implemented. Further, multi-agent cooperation strategies were investigated in tandem with the original purpose.
- **Role:** As the sole author of the thesis, I produced all additions to the original environment, including heavy adaptations, tests and benchmarks.
- **Status:** Public at https://github.com/asbjornhp/MAVisPO

#### [Impact of post-processing on low-level artifacts in synthetically generated images]
- **Concept:** As images circulate online, they naturally undergo a series of transformations such as compression, cropping, and resizing. This project studies how post-processing steps impact the performance of Synthetical Image Detection (SID) models, focusing on the degradation of low-level artifacts and their impact on synthetic image detection.
- **Role:** As a group project, I was the main contributor to setting up the deep learning models from SIDbench (https://github.com/mever-team/sidbench), and writing the Bash Script, to benchmark chosen models through a data-processing pipeline, on DTUs computer clusters.
- **Status:** Public at https://github.com/asbjornhp/Deeplearning-Group16

#### [Hosting an Image Classifier through Google Cloud Platform]
- **Concept:** To gain knowledge of proper CI/CD pipelines, cloud hosting and Docker, a finetuned deep learning model was created, containerized and hosted through GCP. A custom made frontend and FastAPI calls were developed, for both usage and performance monitoring. Testing of server load was done as well. The developed frontend can be found at https://frontend-739688782639.europe-west1.run.app/ .
- **Role:** In this group project, I was the main contributor in everything that happened remotely through Cloud Run on the GCP, containerization, and the development of the FastAPI calls necessary for monitoring.
- **Status:** Public at https://github.com/iasonrap/mlops31

#### [Semantic Correlation of News and Stock Trends]
- **Concept:** By using webscrapers to gather recent news articles and social media posts, we performed semantic analysis and clustering to investigate if a correlation exists between positive public perception and an increase in stock trends for the leading tech-firms. To do this, we used BERT, common data cleaning techniques and various clustering methods.
- **Role:** I performed the role of data analyst in this project, being the main driver behind the clustering of data points and the correlation analysis.
- **Status:** Public at https://github.com/FenrisWulven/StockMarketNewsImpact

### [BugNIST Classification: 2D versus 3D]
- **Concept:** The idea of this project was to benchmark the difference between using 3D and 2D images for classification purposes. For this, we did hyperparameter tuning and training of a ResNET model and benchmarked their performance through Weights and Biases, yielding also nice visualizations. This culminated in a poster which can be viewed.
- **Role:** Together with my teammate, we covered the same tasks with orchestrating the training loop and data processing pipeline.
- **Status:** Public at  https://github.com/asbjornhp/3D_2D_Bug_Classify
